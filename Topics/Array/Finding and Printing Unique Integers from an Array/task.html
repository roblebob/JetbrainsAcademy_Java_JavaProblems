Assume that all numbers used will be integers and greater than 0. Write a program that finds the unique elements in a given array of integers and prints them out in the sequence they appear. Your program will first take an integer n, the total number of integers in the array. Next it will take the n integers as input. It should print the unique integers, each on a new line, ignoring duplicates. If no integer is unique, it should print 'No unique integers'.
Sample Input:
5
1
2
2
3
1

Sample Output:
3


Sample Input:
4
7
7
7
7

Sample Output:
No unique integers



Memory limit: 256 MB
Time limit: 5 seconds
